# Zircon FRT

**OSINT Web Portal for Automated Data Collection, Storage, Search & Monitoring**

> Phase 1 setup in progress...

## About

Zircon FRT is a web portal for automated collection, storage, search, and monitoring of data for OSINT activities. It provides a unified workspace for cybersecurity specialists, OSINT analysts, and brand protection teams.

## Tech Stack

- **Backend:** Python 3.12, FastAPI, Celery, PostgreSQL, Elasticsearch, Redis
- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Infrastructure:** Docker, Nginx, ClamAV

## Quick Start

```bash
docker-compose up --build
```
